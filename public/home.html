Bem-vindo ao SEPA

Como fazer login

  1. Informe seu e-mail cadastrado no campo "Email"
  2. Digite sua senha no campo "Senha"
  3. Marque "Mostrar senha" se quiser visualizar o que está digitando
  4. Clique no botão "Entrar" para acessar o sistema
  5. Caso tenha esquecido sua senha, clique em "Esqueceu senha?"

Como se cadastrar

  1. Clique em "Criar sua Conta" abaixo do botão de login
  2. Selecione seu tipo de usuário (Docente ou Administrador)
  3. Preencha todos os campos obrigatórios:
    • Nome completo
    • E-mail válido
    • Telefone para contato
    • Senha segura
  4. Marque "Mostrar senha" para verificar se digitou corretamente
  5. Clique em "Cadastrar" para finalizar

Recuperação de Senha

  1. Clique em "Esqueceu senha?" na tela de login
  2. Informe o e-mail cadastrado no sistema
  3. Clique em "Enviar Link de Recuperação"
  4. Verifique sua caixa de entrada (e spam) para encontrar o e-mail com as instruções
  5. Siga o link no e-mail para criar uma nova senha
